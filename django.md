## 장고 튜토리얼

장고는 MTV (Model — Template — View) 을 하게 도와주는 웹 프레임워크


### 시작하기

1. 폴더 만들어주기
2. 가상환경 만들기, 실행
3. 터미널에서 django 설치
4. django-admin 으로 기본적인 앱 설치

### Model
5. setting.py에 앱 추가
6. model.py에서 데이터 필드와 속성 정하기
7. manage.py를 실행시켜 makemigrations, migrate 두 명령어로 DB 생성
8. sqlite와 어떻게 연동시킬지는 더 찾아봐야 함.
